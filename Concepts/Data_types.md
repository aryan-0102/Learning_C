># Data Types

* ### INT 
  Integer data type stores values like 1,22,456, etc. <br>
  It occupies 4 bytes and has a range -32767 to 32767. <br>
  Syntax : int variable_name ;

* ### CHAR
  Character data types is used to store a single alphabet .<br>
  It is stored in single quotes and takes 1 byte with a range -127 to 127 <br>

* ### FLOAT
  It stores values with upto 6 places after decimal.
  These require 4 bytes to store.

* ### DOUBLE
  It also stores values with up to 10 digits after decimal.<br>
  It takes 8 bytes to store.
